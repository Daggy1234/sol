Overview
==========

Sol is a library that binds C++ to lua and vice versa. It's a header-only library that requires no dependencies. It aims
to support Lua 5.2 and higher and to make binding C++ to lua as easy as it can be. Sol requires a C++11 compatible
compiler. The officially supported compilers are as follows:

* Clang 3.4
* GCC 4.8
* GCC 4.9

Users of the library are encouraged to read the examples provided in the example/ directory when cloning the repository.
